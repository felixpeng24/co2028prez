Voice & Tone

We're talking to our classmates, not writing a corporate memo. Keep it real.

Do

  • Write like you're texting a group chat (but readable)
  • Use emojis when it fits the vibe
  • Bold important names, prices, dates
  • Keep sentences short and punchy
  • Start with hype if the news is exciting
  • Use "we" and "our" — it's a class thing
  • Be direct about what people need to do

Don't

  • Sound like a formal announcement
  • Use "Dear Class of 2028" or "Greetings"
  • Write walls of text with no formatting
  • Bury the important info in paragraphs
  • Be vague about deadlines or next steps
  • Forget to include payment/form/RSVP details

Email Structure

The standard layout for class emails. Not every email needs every section, but this is the go-to order.

Section Required? Notes
Greeting Always HELLO 2028:
Headline Always One-liner about what this email is about
Context / Body Usually 1-3 sentences max. Get to the point.
Image If relevant Merch, flyers, event photos
Details If applicable Price, dates, location — use section labels + detail rows
Action / CTA Usually Form link, RSVP, sign-up — make it obvious
Payment Info If applicable Use the info box component
Sign-off Always Friendly close + name + title
